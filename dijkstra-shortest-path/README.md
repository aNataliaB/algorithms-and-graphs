# Shortest Path Finder using Dijkstra's Algorithm
A pure Python interactive application that implements Dijkstra's algorithm to find the shortest path and minimum cost between vertices in a weighted graph.

## Project Overview
The application showcases a fundamental understanding of graph theory and algorithm design. It features a predefined weighted graph structure and provides an interactive command-line interface (CLI) that allows users to dynamically query optimal paths between different nodes.

### Key Features:
* **Algorithmic Core:** Pure Python implementation of Dijkstra's algorithm tracking distances, visited states, and path history.
* **Interactive CLI:** A user loop that collects node inputs, validates them against existing graph vertices, and prints out formatted route layouts (e.g., `A -> B -> C -> D`).
* **Robust Error Handling:** Detects invalid node inputs and handles edge cases such as disconnected nodes, informing the user when no valid path exists.
* **Complexity Analysis:** Theoretical analysis confirming a time complexity of O(V^2), optimizing efficiency for the chosen matrix/dictionary structure.
